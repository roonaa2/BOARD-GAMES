Detailed Description

Represents the Numerical Tic-Tac-Toe game board.

This class inherits from the generic Board<char> class and implements the specific logic required for the Numerical Tic-Tac-Toe game. Player 1 uses odd numbers (1,3,5,7,9), Player 2 uses even numbers (2,4,6,8). The goal is to make a row, column, or diagonal sum to exactly 15.
