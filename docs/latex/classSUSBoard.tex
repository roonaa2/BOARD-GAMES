\doxysubsection{Detailed Description}
Represents the SUS Tic-\/\+Tac-\/\+Toe game board. 

This class inherits from the generic {\ttfamily \doxylink{classBoard}{Board}\texorpdfstring{$<$}{<}char\texorpdfstring{$>$}{>}} class and implements the specific logic required for the SUS game where players form "{}\+S-\/\+U-\/\+S"{} sequences.
