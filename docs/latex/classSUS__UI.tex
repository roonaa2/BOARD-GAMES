\doxysubsection{Detailed Description}
User Interface class for the SUS Tic-\/\+Tac-\/\+Toe game. 

Inherits from the generic {\ttfamily \doxylink{classUI}{UI}\texorpdfstring{$<$}{<}char\texorpdfstring{$>$}{>}} base class and provides SUS-\/specific functionality for player setup and move input.
