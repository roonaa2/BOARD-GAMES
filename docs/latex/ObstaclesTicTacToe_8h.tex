Game Rules\+:
\begin{DoxyItemize}
\item A 6x6 grid
\item After every round (one turn for each player), two new obstacle cells are randomly added
\item Obstacles (\#) cannot be used by either player
\item First player to align four marks in a row (horizontally, vertically, or diagonally) wins
\item The game is a draw if the board fills without a winner 
\end{DoxyItemize}
